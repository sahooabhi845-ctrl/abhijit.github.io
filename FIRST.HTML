What is HTML ?

  • Stands for "Hypertext markup language".
  • HTML is the code that is used to structure a webpage and its content.
  • The component is used to design the structure of website are called html tag.
  • HTML (HyperText Markup Language) is the foundational, standard markup language for creating and structuring content on the World Wide Web, using tags and elements to define parts like headings, paragraphs, links, and images, forming the essential backbone that browsers interpret to display all web pages, from simple sites to complex applications, making it the universal language of the internet.

Structure of HTML:

Tags of HTML:

  • Tags are always written inside angel bracket (<>).
  • HTML tags are categorised in two types.
  • These two types of tags are as following:

    1. Paired tag
    2. Unpaired tag

1.Paired tag:

  • Paired tag he tag which has bois other wise known as container tag.
  • The tag which has both opening tag and closing tag , that tag is known as paired tag/container tag.
  • Opening tag - <tag name>
  • Closing tag - </tag name>
  • Examples of paired tags:
    • <html></html> , <title></title> , <head></head> , <body></body> , <p></p>

2.Unpaired tag:

  • Unpaired tag is other wise known as standalone tag.
  • The tag which has no any closing tag only opening tag , that tag is known as unpaired tag/standalone tag.
  • Only opening tag - <tag name>
  • Examples of unpaired tags:
    • <br> , <hr> , <vr>

An example of HTML code:

    <!DOCTYPE html>
    <html>
    <head>
    <title>My First Web Page</title>
    </head>
    <body>
    <h1>Welcome to My First HTML Page</h1>
    <p>This is my first HTML program!</p>
    </body>
    </html>
